# Toggle Drone

A mod that introduces the **Adaptive Nanodrone System** (`SYSTEM_DND_TOGGLE`), allowing pilots to switch between stopping objects (Tug) and retrieving them (Haul) on the fly.

> [!WARNING]
> Requires HevLib to function
> https://github.com/rwqfsfasxc100/HevLib/releases

## Features

*   **Dynamic Mode Switching:** Toggle between **Haul** and **Tug** using your slot activation key.
*   **Default Mode Tuning:** Set the preferred starting mode for each ship slot via the Tuning menu at Enceladus Prime.


## Configuration

*   **Cycle Order**:
    *   **Fixed**: Always cycles `Haul -> Tug -> Off`.
    *   **Adaptive**: Cycles `Default Mode -> Other Mode -> Off`.
